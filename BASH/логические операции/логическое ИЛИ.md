возвращает true, если один из операндов истинен, используется в операциях [[проверка условий]]

применяется внутри одинарных [[квадратные]] скобках с помощью опции -o ([[test]]) или двойных с помощью двойного [[прямой]] слеша (||)

может использоваться в построении ИЛИ-списков (следующая команда выполнится только если предыдущая завершилась с ненулевым кодом)
 command1 || command2
 
 оператор && может использоваться и в арифметических вычислениях
  echo $(( 1 && 2 )) $((3 && 0)) $((4 || 0)) $((0 || 0)) \# 1 0 1 0
