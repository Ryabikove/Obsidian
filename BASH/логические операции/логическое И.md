возвращает true, если оба из операндов истинны, используется в операциях [[проверка условий]]

применяется внутри одинарных [[квадратные]] скобках с помощью опции -a ([[test]]) или двойных с помощью двойного [[амперсанд]] (&&)

может использоваться в построении И-списков (следующая команда выполнится только если предыдущая завершилась с нулевым кодом)
 command1 && command2

оператор && может использоваться и в арифметических вычислениях
 echo $(( 1 && 2 )) $((3 && 0)) $((4 || 0)) $((0 || 0)) \# 1 0 1 0
