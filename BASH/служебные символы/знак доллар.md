[[подстановка значения переменной ]]
 echo $a \# вывод значения переменной "а"

в [[регулярные выражения]] означает конец строки
 grep -v "^$" file \# вывод содержимого файла, кроме пустых строк

в сочетании с [[фигурные]] скобки подстановка параметра
 ${var:-20} \# подстановка значения переменной var, если её нет, "20"

в сочетании с [[звездочка]] или [[собачка]] все значения [[параметры командной строки]]: $\* $@

в сочетании с [[знак вопроса]] [[код завершения]] последней выполненной команды, функции или сценария: $?

в сочетании с нижним подчеркиванием \$_ образует специальную переменную, которая содержит последний аргумент предыдущей команды.

двойной знак хранит id процесса сценария ([[PID]])
