\# может представлять из себя:
 1. [[комментарий]]. 
	(1. echo "ok" \#comment)

 2. символ для нахождения кол-ва символов, которые содержит [[переменная]] или один из эл-тов [[массив]]а 
	 (${#var} ${#list\[n]})
 
 3. кол-во элементов в [[массив]]е 
	 (${#list\[@]} или ${#list\[\*]})

 4. число, в записи которого присутствует символ \#, расценивается как запись числа с указанием основы счисления в виде ОСНОВА#ЧИСЛО.
	echo "32\#77"              # 231
