внутри круглых скобок заключаются группы команд, исполняемые в дочернем процессе ([[subshell]])
 (a=hello; echo $a)

так же используются для инициализации [[массив]]ов 
 a=(arg1 arg2 arg3)

в сочетании со [[знак доллар]] может использоваться для подстановки вывода команд, заключенных в круглые скобки, аналогично [[обратные]] кавычки
 a=$(grep "arg" file) \# значение переменной равно выводу команды grep

внутри двойных круглых скобок вычисляются целочисленные выражения
 ((10/2))
