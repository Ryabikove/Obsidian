используется для подстановки значений в сочетании с символом [[знак доллар]]
 ${var:-20} \# подстановка значения переменной var, если её нет, "20"

так же используется для перечисления различных значений в одной команде:
 cat file{1,2,3*} \# вывод файлов file1, file2, file3*
 данная команда интерпретируется как список команд, разделенных точкой с запятой
 ==использование неэкранированных пробелов внутри скобок недопустимо==

используется для создания блока кода ([[вложенный блок]]), фактически создавая анонимную функцию, ==однако функцией этот блок не является и переменные в таких блоках доступны объемлющему сценарию==
 { a=123; echo $a > file}
 не вызывает запуск дочернего процесса ([[subshell]])

в сочетании с [[обратный]] слеш и [[точка-с-запятой]], образует аналог [[pathname]], полного имени файла
 {} \\;
