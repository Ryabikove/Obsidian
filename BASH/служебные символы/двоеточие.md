пустая команда, эквивалент операции [[NOP]]

может рассматриваться как эквивалент [[встроенная команда]] [[true]]

является [[встроенная команда]]

этот символ можно использовать в качестве заполнителя для [[условные операторы]] [[if]], [[while]]
 while : ; do
	if : ; then
		echo ok
	fi
  done

операций, предполагающих наличие двух операндов
 : ${username=\`whoami\`}

в операциях [[замена подстроки]] с [[подстановка значения переменной]]

в комбинации с оператором ">" (: > file) усекает длину файла до 0, если указан несуществующий файл, то он будет создан (эквивалент [[touch]])
 
 то же самое, что и cat /dev/null > file, однако не производится создание нового процесса, т.к. ":" встроенная команда в комбинации с ">>"
 
 применимо только к обычным файлам и не применимо к конвейерам ([[прямой]] слеш), [[символическиe]] [[ссылки]] и другим спец файлам

может использоваться для [[комментарий]]
 ": Это комментарий, но он генерирует сообщение об ошибке (if \[ $x -eq 3 ])"

разделитель полей в различных конфигурационных файлах и переменной PATH ([[встроенные переменные]])
