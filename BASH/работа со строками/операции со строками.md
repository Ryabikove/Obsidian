Одни операции являются подмножеством операций [[подстановка значения переменной]], а другие -- совпадают с функциональностью команды UNIX - [[expr]].

string=abcABC123ABCabc
 
поиск длины целой строки:
 - \${\#string}
	 echo \${\#string}          \# 15
 - expr length \$string
	 echo \`expr length \$string\`    \# 15
 - expr "\$string" : '.\*
	echo \`expr "\$string" : '.\*'\`   \# 15
поиск длины подстроки в строке:
 ==подсчет совпадающих символов ведется с начала строки==
 
 ==\$substring - регулярное выражение==
 - expr match "\$string" '\$substring'
	echo \`expr match "\$string" 'abc\[A-Z]\*.2'\`   \# 8
 - expr "\$string" : '\$substring'
	echo \`expr "\$string" : 'abc\[A-Z]\*.2'\`    \# 8
поиск позиции первого совпадения:
 - expr index \$string \$substring
	 echo \`expr index "\$string" C12\`   \#6 позиция символа C
	 echo \`expr index "\$string" 1c\`  \#3 символ 'c' совпал раньше, чем '1'
извлечение подстроки из строки:
 - \${string:position} - извлекает подстроку из string, начиная с позиции position.
	 echo \${string:7}        \# 23ABCabc
	 
	 echo \${string:-4}      \# abcABC123ABCabc
	 по-умолчанию выводится полная строка
	 
	 ==круглые скобки или дополнительный пробел "экранируют" параметр позиции==
	 echo \${string:(-4)}         \# Cabc
	 echo \${string: -4}          \# Cabc
	 ==если строка string - "\*" или "@", то извлекается позиционный парамер (аргумент), с номером position==
 - "\${string:position:length}" - извлекает length символов из string, начиная с позиции position
	echo \${string:7:3}          \# 23A
	
	==если string - "\*" или "@", то извлекается до length позиционных параметров (аргументов), начиная с position==
 - "expr substr $string $position $length" то же, что и \${string:position:length}
	 echo \`expr substr \$string 1 2\`   \# ab
 - expr match "\$string" '\\(\$substring\\)' - находит и извлекает первое совпадение substring в string
	 ==substring - это регулярное выражение==
	 
	 echo \`expr match "$string" '\\(.\[b-c]\*\[A-Z]..\[0-9]\\)'\`   \# abcABC1
 - expr "\$string" : '\\(\$substring\\)' - находит и извлекает первое совпадение substring в string
	 ==substring - это регулярное выражение==
	 
	 echo \`expr "\$string" : '\\(.\[b-c]\*\[A-Z]..\[0-9]\\)'\`     \# abcABC1
 - expr match "\$string" '.\*\\(\$substring\\)' - находит и извлекает первое совпадение substring в string. поиск начинается с конца string.
	 ==substring - это регулярное выражение==
	 
	 echo \`expr match "\$string" '.\*\\(\[A-C]\[A-C]\[A-C]\[a-c]\*\\)'\`    \# ABCabc
 - expr "\$string" : '.\*\\(\$substring\\)' - находит и извлекает первое совпадение substring в string. поиск начинается с конца string.
	==substring - это регулярное выражение==
	
	echo \`expr "\$string" : '.\*\\(......\\)'\`                 \# ABCabc
