проверяет - является ли [[код завершения]] списка команд 0, если верно, выполняет блок кода за словом then, если неверно - else.

возможный синтаксис:
 if grep -q "lol" file1; then
	echo ok
 elif echo lol >file1; then
	echo ok
 else
	echo notok
 fi
 if \[...]
 then
 if \[\[...]]; then
 if $(...); then
 if ((...)); then
 if test ...
 then

допускает наличие вложенных проверок:
 if ...; then
	if ...; then
	else
	fi
 else
	if ...; then
	else
	fi
 fi

==Когда if и then располагаются в одной строке, то конструкция if должна завершаться точкой с запятой.==

==if, и then - это [[зарезервированные слово]]. Зарезервированные слова начинают инструкцию, которая должна быть завершена прежде, чем в той же строке появится новая инструкция==
