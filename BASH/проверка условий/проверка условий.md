для проверки условий, применяются:
 - команда [[test]]
 - различного вида скобочные операторы ([[квадратные]] и [[круглые]] скобки)
	[[круглые]] скобки (( ... )) и предложение [[let]] возвращают код 0, если результатом арифметического выражения является ненулевое значение.
	
	условие, находящееся внутри квадратных скобок может быть проверено без использования оператора **if**
 - условный оператор [[if]]/then
	 конструкция "if COMMAND" возвращает код возврата команды COMMAND
