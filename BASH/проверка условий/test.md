воспринимает свои аргументы как выражение сравнения или как файловую проверку, возвращая код завершения (0 - истина, 1 - ложь)

[[встроенная команда]] Bash, которая выполняет проверки файлов и производит сравнение строк.
 ==не вызывает внешнюю (/usr/bin/test) утилиту==

опции сравнения файлов:
 - -e - файл существует
 - -f - обычный файл (не каталог и не файл устройства)
 - -s - ненулевой размер файла
 - -d - файл является каталогом
 - -b - файл является блочным устройством (floppy, cdrom и т.п.)
 - -c - файл является символьным устройством (клавиатура, модем, звуковая карта и т.п.)
 - -p - файл является каналом
 - -h - файл является символической ссылкой
 - -L - файл является символической ссылкой
 - -S - файл является сокетом
 - -t - файл [[дескриптор]] связан с терминальным устройством
	 Этот ключ может использоваться для проверки -- является ли файл стандартным устройством ввода stdin (\[ -t 0 ]) или стандартным устройством вывода stdout (\[ -t 1 ]).
 - -r - файл доступен для чтения
 - -w - файл доступен для записи
 - -x - файл доступен для исполнения
 - -g - set-group-id (sgid) флаг для файла или каталога установлен ([[флаги файлов]])
 - -u - set-user-id (suid) флаг для файла установлен ([[флаги файлов]])
 - -k - флаг sticky bit (бит фиксации) установлен ([[флаги файлов]])
 - -O - вы являетесь владельцем файла
 - -G - вы принадлежите к той же группе, что и файл
 - -N - файл был модифицирован с момента последнего чтения
 - -nt - файл f1 новее f2 (f1 -nt f2)
 - -ot - файл f1 старее f2 (f1 -ot f2)
 - -ef - файлы f1 и f2 [[жесткая]] ссылки на один и тот же файл (f1 -ef f2)
 - ! - "НЕ" -- [[логическое НЕ]] (инверсия) результатов всех вышеприведенных проверок ([[восклицательный знак]])
	 ==возвращается true если условие отсутствует==

опции сравнения целых чисел:
 внутри [[квадратные]] скобок
  - -eq -  равно \[ "\$a" -eq "\$b" ]
  - -ne - не равно
  - -gt - больше
  - -ge - больше или равно
  - -lt - меньше
  - -le - меньше или равно
 внутри [[круглые]] скобок:
  - < - меньше (("\$a" < "\$b"))
  - <= - меньше или равно
  - > - больше
  - >= - больше или равно

опции сравнения строк:
 - = - равно \[ "\$a" = "\$b" ]
 - == - равно
	 синоним оператора = (можно использовать в двойных [[квадратные]] скобки)
 - != - не равно
 - < - меньше, в смысле величины ASCII-кодов
	 ==для использования внутри одинарных квадратных скобок необходимо экранировать==
 - > - больше, в смысле величины ASCII-кодов
	==для использования внутри одинарных квадратных скобок необходимо экранировать==
 - -z - строка "пустая", т.е. имеет нулевую длину
 - -n - строка не "пустая"
	 ==оператор -n требует, чтобы строка была заключена в кавычки внутри квадратных скобок==
	 ==проверка строк, не заключенных в кавычки, происходит оператором ! -z, или просто указание строки без кавычек внутри квадратных скобок==

условные опции:
