или character varying()
символьный тип данных строка переменной длины.

имеет синтаксис
 val varchar(n) - где n целое число от 1 до 10485760 включительно
 без указания длины принимаются строки любой длины

в отличие от char, строка дополняться пробелами не будет, а n ограничивает строку только по максимальной длине.
если строка длиннее n, то данные не запишутся и вернется ошибка, если только все лишние символы не являются пробелами (тогда они будут усечены до максимально допустимой длины).

==может хранить до 65535 байт информации==

Начиная с MySQL 5.6 типы char и varchar по умолчанию используют кодировку UTF-8
