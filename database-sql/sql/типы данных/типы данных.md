для каждого столбца в базе данных необходимо указать свой тип.

==некоторые базы данных могут не поддерживать определенные типы данных==

тип данных определяет:
 - какие значения могут храниться в столбце
 - сколько они могут занимать места в памяти

типы данных делятся на:
 - [[символьные данные]]
 - [[числовые данные]]
 - [[sql/типы данных/денежные типы|денежные типы]]
 - [[данные даты и времени]]
 - [[составные типы данных]]
 - [[sql/типы данных/бинарные/бинарные данные|бинарные данные]]
