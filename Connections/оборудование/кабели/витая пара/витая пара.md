кабель, использующийся для соединения уст-в между собой в локальной сети

представляет из себя 4 перекрученные пары по 2 проводника с изоляцией определенного цвета

для соединения уст-в в сети ethernet используется различна обжимка для коннектера [[8P8C]] настроенного по стандарту [[RJ]]:
 - при соединении уст-в одного уровня сети - [[прямая]] обжимка
 - при соединении уст-в разного уровня - [[перекрестная]] обжимка
