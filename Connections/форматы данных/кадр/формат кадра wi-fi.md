при высокой вероятности возникновения ошибки при передаче кадров, они фрагментируются.

разделяют несколько типов кадров в wi-fi:
 - данных
 - контроля (служебные, RTS, CTS, ACK)
	[[maca]]
 - управления 
	используются для реализации различных сервисов wi-fi

При передаче в распределительную систему и из неё порядок адресов меняется

к распределительной среде:
 - Адрес 1 - Receiver address адрес уст-ва, принимающего данные из беспроводной среды /Transmitter address адрес уст-ва, передающего данные в беспроводную среду
 - Адрес 2 - Source address адрес отправителя
 - Адрес 3 - Destination address адрес получателя

от распределительной среды:
 - Адрес 1 - Receiver address/Destination address
 - Адрес 2 - Transmitter address
 - Адрес 3 - Source address

передача кадра в одноранговом режиме:
 - Адрес 1 - RA/DA
 - Адрес 2 - TA/SA
 - Адрес 3 - BSSID (идентификатор сети)

беспроводной мост:
 - Адрес 1 - RA
 - Адрес 2 - TA
 - Адрес 3 - DA
 - Адрес 4 - SA
