2-ой уровень взаимодействия, предает сообщения(кадры) по каналу связи, определяя начало и конец сообщения в потоке бит

при получении пакета данных от вышестоящего уровня добавляет к нему заголовок и концевик

способы определения границ каждого кадра:
 -  указание длины в байтах в начале каждого кадра (не применяется на практике)
 -  начало и конец каждого кадра отмечаются спец последовательностями байтов или бит (протоколы [[BSC]], [[HDLC]], [[PPP]])
 - ![[преамбула]]
 - избыточное кодирование - начало кадра символы J(11000)K(10001), конец кадра - символ Т(01101)


дополнительно обеспечивает обнаружение и коррекцию ошибок
 - обнаружение происходит с помощью контрольной суммы (актуально для сети с редкими ошибками)
  актуально для сети с частыми ошибками (v)
 - исправление происходит с помощью избыточной информации, прикрепленной к данным, позволяющей обнаруживать и исправлять ошибки
 - так же исправление происходит с помощью повторной отправки сообщения (в этом случае отправитель ждет подтверждения о получении, без него по таймауту сообщение будет выслано повторно)

методы повторной отправки:
 - остановка + ожидание (ждет подтверждения от получателя)
 - скользящее окно - передача нескольких кадров(окна) не дожидаясь подтверждения, получатель подтверждает получение только последнего кадра (используется в основном в протоколе [[TCP]])

канальный уровень разделен на два подуровня:
 - управления логическим каналом ([[LLC]]) - отвечает за передачу данных (создание кадров, обработку ошибок и т.д.), является общим для разных технологий
 - управления доступом к среде ([[MAC]]) - используется только если технология с разделяемым доступом, отвечает за адресацию, совместное использование среды, является специфичным для разных технологий

==в широковещательной сети управляет доступом к среде передачи данных (1 уст-во одновременно) и отвечает за физическую адресацию==

Для некоторых типов каналов с множественным доступом к среде передачи данных обеспечивает:
 - адресацию 
 - согласованный доступ к каналу

включает в себя технологии:
 - [[ethernet]]
 - [[wi-fi]]
  
Единица передаваемых данных - [[кадр]]
Оборудование - [[коммутатор]], [[точка доступа]]
