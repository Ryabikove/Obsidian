Наиболее ближайший уровень к среде передачи данных обеспечивает просто передачу данных, не вникая в их содержание 

Вышестоящие уровни могут решать уже более сложные задачи, например поиск маршрута в составной сети

Каждый уровень имеет несколько атрибутов:
- Сервис - описывает какие функции реализует уровень
- Интерфейс - набор примитивных операций, которые нижний уровень предоставляет верхнему
- Протокол - правила и соглашения, используемые для связи уровня N одного уст-ва с уровнем N другого

уровни разных уст-в не имеют возможности взаимодействовать друг с другом, кроме взаимодействующих с физической средой
