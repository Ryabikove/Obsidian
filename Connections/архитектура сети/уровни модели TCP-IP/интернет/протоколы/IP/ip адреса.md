адреса протокола [[IP]], используемые для уникальной идентификации компьютеров в составной сети

в зависимости от версии протокола длина адреса различается:
 - 4 байта для IPv4
 - 16 байт для IPv6

==IPv4==
адрес этой версии представляет из себя 4 десятичных числа от 0 до 255, разделенных точками
 213.180.193.3 11010101.10110100.11000001.00000011

старшая часть адреса - адрес подсети
младшая часть адреса - адрес уст-ва в подсети

для разделения сетевой и хостовой части адреса используют маску подсети:
 1. маска подсети и ip адрес переводятся в бинарный вид
 2. маска и адрес складываются методом _И_
	11111111.00000000.00000000.00000000 (255.0.0.0)
	 _И_
	11010101.10110100.11000001.00000011 (213.180.193.3)
 3. полученный адрес является адресом сети
	11010101.00000000.00000000.00000000 (213.0.0.0)
