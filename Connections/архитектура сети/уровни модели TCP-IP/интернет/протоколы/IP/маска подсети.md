представляет из себя 32 битную последовательность нулей и единиц, разделенных точками по 8 бит.
используется для выделения адреса сети из ip адреса оконечного уст-ва

в маске подсети единицы идут подряд, они не могут чередоваться с нулями, оставшееся пространство заполняется нулями

может иметь вид записи (нижеуказанные записи эквивалентны):
 - /8
 - 11111111.00000000.00000000.00000000
 - 255.0.0.0
