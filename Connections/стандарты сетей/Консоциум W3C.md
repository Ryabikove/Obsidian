World Wide Web Consociety принял стандарты в области WWW

Все документы являются лишь рекомендациями, что отражено в их названии

Наиболее популярные рекомендации W3C:
 - язык разметки HTML
 - таблица стилей CSS
 - архитектура Web-сервисов
 - язык разметки XML

Все рекомендации доступны бесплатно по адресу:
https://www.w3.org/standards/
