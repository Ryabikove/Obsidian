специальный постфикс для числовых типов данных, включающий в себя диапазон от 0 до n, где n это сумма модулей классического типа
пример:
 smallint - диапазон от -32768 до 32767
 smallint unsigned - диапазон от 0 до 65535 (32768+32767)
 tinyint - диапазон от -128 до 127
 tinyint unsigned - диапазон от 0 до 255 (127+128)

может быть использован вместе с типами:
 - [[tinyint]]
 - [[sql/типы данных/числовые/smallint|smallint]]
 - [[mediumint]]
 - [[int]]
 - [[bigint]]
