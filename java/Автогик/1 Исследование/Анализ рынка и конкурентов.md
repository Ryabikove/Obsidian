**"За рулем"**
Преимущества:
- Реализован большой объём информации об автомобилях;
- Реализована общая информация об уст-вах автомобилей
Недостатки:
- Некрасивый интерфейс;
- Устаревшие данные;
- Нет возможности сравнивать;


**Автопрагмат** 
Преимущества:
- Приятный интерфейс;
- Практически актуальная информация;
- Есть информация о достоинствах и недостатках авто;
- Собраны рекомендации по конфигурациям и оценка по категориям;
- Собраны комментарии от автовладельцев;
- Есть тест на выбор авто - идея хорошая;
- Реализован калькулятор стоимости владения авто в год;
Недостатки:
- Очень ограниченный набор автомобилей;
- Реализация теста на выбор авто слабая;
- Не являются платформой для продажи авто


**Авто.ру** 
Преимущества:
- Очень привлекательный и удобный интерфейс
- Есть мобильное приложение
- Есть авторизация и бонусные функции от неё
- Реализован журнал с новостями
- Есть калькулятор расчета стоимости авто
- Есть возможность проверки истории автомобиля
- Есть личные кабинеты для продавцов авто и привозящих из-за границы
- Есть возможность сравнения авто
- Есть справочная информация о моделях и конфигурациях
Недостатки:
- Нет модерации на соответствие содержимого карточки теме

**Какую конкретную проблему мы решаем?** - позволяем быстро сравнить по различными характеристикам автомобили и подобрать интересующую модель

**Почему пойдут к вам, а не останутся у конкурентов?** - потому что у других конкурентов нет такой жесткой модерации карточек
