Окно с перечнем рекомендованных моделей авто.

Принципы:
- Представляет из себя [[Автогик/3 Информационная архитектура/Карты/Дополнительные сущности/Список сущностей|список сущностей]]: моделей и поколений.
- Если пользователь не авторизованный, то содержит информацию из [[Автогик/3 Информационная архитектура/Карты/Модули/Популярные|списка популярных]]
- Рекомендации собираются в приоритете из следующих:
	- Избранное
	- Просмотры (последние 3 месяца)
- Список упорядочен автоматически, изменить нельзя

В телеграм-боте вызывается командой - /recomended
