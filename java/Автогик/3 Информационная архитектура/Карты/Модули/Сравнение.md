Окно, содержащее добавленные в сравнительный лист модели/объявления.

Окно в виде таблицы, где сравниваются все характеристики объектов сравнения, разделенные на следующие группы:
- Приобретенные (цена)
- Основные
- Кузов
- Интерьер
- Двигатель
- Подвеска и тормоза
- Прочее
- Цена владения в год - берется из [[Автогик/3 Информационная архитектура/Карты/Модули/Калькуляторы|калькулятора]]

Принципы сравнения:
- Сущности разных уровней между собой невозможно сравнить - они добавляются в разные сравнительные таблицы
- Бренды и концерны нельзя сравнить

Присутствует функция ИИ помощника, который делает выводы по каждой категории отдельно, и по всем в совокупности.

В телеграм-боте вызывается командой - /compare
