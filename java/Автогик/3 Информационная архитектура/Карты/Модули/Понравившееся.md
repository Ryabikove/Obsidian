Окно, содержащее [[Автогик/3 Информационная архитектура/Карты/Список сущностей|список сущностей]], которые были отмечены пользователем.

Задуманы как список сущностей, которые пользователь выбирает временно с целью покупки или сравнения.

Принципы:
- В список могут быть добавлены только модели, поколения, карточки

В телеграм-боте вызывается командой - /liked
