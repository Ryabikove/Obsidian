Окно с регистрацией или входом в текущий аккаунт.

Принципы:
- На внешнем сайте происходит регистрация или авторизация. 
- Авторизация открывает больше возможностей для пользователя.
- Неавторизованные пользователи имеют доступ к базовому функционалу продукта.

При регистрации необходимо:
- Указать логин
- Пароль дважды
- Способ авторизации (почта или телефон)
- Пол
- Дату рождения
- Город проживания

При авторизации необходимо указать:
- Тип авторизации логин/номер телефона/почту
- Пароль/проверочный код (только для номера телефона или почты)


Переход к модулю осуществляется:
- На сайте:
	- Через [[Автогик/3 Информационная архитектура/Карты/Модули/Дополнительные/Основное меню|основное меню]]
- В телеграм-боте:
	- С помощью команды "/login"
