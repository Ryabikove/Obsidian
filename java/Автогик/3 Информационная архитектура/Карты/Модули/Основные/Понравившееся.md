Окно, содержащее [[Автогик/3 Информационная архитектура/Карты/Модули/Дополнительные/Список сущностей|список сущностей]], которые были отмечены пользователем.

Задуманы как список сущностей, которые пользователь выбирает временно с целью покупки или сравнения.

Принципы:
- В список могут быть добавлены только модели, поколения, карточки.

Переход к модулю осуществляется:
- На сайте:
	- Через [[Автогик/3 Информационная архитектура/Карты/Модули/Дополнительные/Основное меню|основное меню]].
- В телеграм-боте:
	- С помощью команды "/liked".
