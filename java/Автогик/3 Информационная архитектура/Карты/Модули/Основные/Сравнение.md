Окно, содержащее добавленные в сравнительный лист модели/объявления.

Окно в виде таблицы, где сравниваются все характеристики из [[Автогик/3 Информационная архитектура/Структура данных/Сущности/Автопром/Комплектация|комплектации]] и приобретенные из [[Автогик/3 Информационная архитектура/Структура данных/Сущности/Автопром/Карточка|карточки]].
К приобретенным характеристикам относятся:
- пробег;
- цена;
- деффекты;
- кол-во владельцев;
- прочее из ПТС;

Принципы сравнения:
- Сущности разных уровней между собой невозможно сравнить - они добавляются в разные сравнительные таблицы.
- Бренды и концерны нельзя сравнить.

Присутствует функция ИИ помощника, который делает выводы по каждой категории отдельно, и по всем в совокупности.

Переход к модулю осуществляется:
- На сайте:
	- Через [[Автогик/3 Информационная архитектура/Карты/Модули/Дополнительные/Основное меню|основное меню]]
- В телеграм-боте:
	- С помощью команды "/compare"
