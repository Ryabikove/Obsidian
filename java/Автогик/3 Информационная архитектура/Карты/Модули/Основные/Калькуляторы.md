Окно, содержащее калькуляторы для расчета стоимости:
- растаможки авто
- владения авто в год

Калькулятор растаможки по принципу - [Калькулятор растаможки автомобилей](https://calcus.ru/rastamozhka-auto)
Калькулятор владения по принципу - [Калькулятор владения авто | Автопрагмат](https://autopragmat.ru/calculator/)

Принципы:
- Для расчета стоимости в обоих калькуляторах показатели можно ввести вручную или выбрать модель/поколение/карточку для автоматического расчета.

Переход к модулю осуществляется:
- На сайте:
	- Через [[Автогик/3 Информационная архитектура/Карты/Модули/Дополнительные/Вспомогательное меню|вспомогательное меню]]
	- Через ссылку в полном представлении [[Автогик/3 Информационная архитектура/Структура данных/Сущности/Автопром/Поколение|поколения]] или [[Автогик/3 Информационная архитектура/Структура данных/Сущности/Автопром/Карточка|карточки]]
- В телеграм-боте:
	- вызывается командой - /halc
	- Через полное представление [[Автогик/3 Информационная архитектура/Структура данных/Сущности/Автопром/Поколение|поколения]] или [[Автогик/3 Информационная архитектура/Структура данных/Сущности/Автопром/Карточка|карточки]]
