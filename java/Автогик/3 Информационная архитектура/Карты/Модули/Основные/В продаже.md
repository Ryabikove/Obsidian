Окно, содержащее [[Автогик/3 Информационная архитектура/Карты/Модули/Дополнительные/Список сущностей|список сущностей]], которые пользователь добавил в продукт.

Принципы:
- Задумывается как модуль, позволяющий добавлять сущности с целью продажи.
- В список могут быть добавлены только [[Автогик/3 Информационная архитектура/Структура данных/Сущности/Автопром/Карточка|карточки]]
- Нельзя добавить сущность в список если аккаунт не является подтвержденным

Переход к модулю осуществляется:
- На сайте
	- Через вспомогательное меню - доступно, только если пользователь прошел авторизацию
	- Через модуль продукта "[[Автогик/3 Информационная архитектура/Карты/Модули/Основные/Об аккаунте|об аккаунте]]" 
- В телеграм-боте:
	- Вызывается командой "/in_sale" - доступно, только если пользователь прошел авторизацию
	- Через модуль продукта "[[Автогик/3 Информационная архитектура/Карты/Модули/Основные/Об аккаунте|об аккаунте]]"
