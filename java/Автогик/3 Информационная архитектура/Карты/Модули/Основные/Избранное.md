Окно, содержащее [[Автогик/3 Информационная архитектура/Карты/Модули/Дополнительные/Список сущностей|список сущностей]], которые были отмечены пользователем.

Принципы:
- Задуманы как список сущностей, которые пользователь выбирает как "авто мечты".
- В список могут быть добавлены только [[Автогик/3 Информационная архитектура/Структура данных/Сущности/Автопром/Модель|модели]] и [[Автогик/3 Информационная архитектура/Структура данных/Сущности/Автопром/Поколение|поколения]]
- Доступно только если пользователь прошел авторизацию

Переход к модулю осуществляется:
- На сайте
	- Через модуль продукта "[[Автогик/3 Информационная архитектура/Карты/Модули/Основные/Об аккаунте|об аккаунте]]" 
- В телеграм-боте:
	- Вызывается командой - /favourite
