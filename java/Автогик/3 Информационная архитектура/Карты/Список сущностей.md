Лента из сущностей в виде кратких представлений.
По нажатию на сущность можно перейти в полное представление сущности и ознакомиться с информацией о ней.
Нельзя в одном окне открыть два полных представления сущности.

Существуют следующие сущности, в порядке от наиболее глобальной, к наиболее частной:
1. [[Автогик/3 Информационная архитектура/Структура данных/Сущности/Автопром/Страна|Страна]]
2. [[Автогик/3 Информационная архитектура/Структура данных/Сущности/Автопром/Концерн|Концерн]]
3. [[Автогик/3 Информационная архитектура/Структура данных/Сущности/Автопром/Бренд|Бренд]]
4. [[Автогик/3 Информационная архитектура/Структура данных/Сущности/Автопром/Модель|Модель]]
5. [[Автогик/3 Информационная архитектура/Структура данных/Сущности/Автопром/Поколение|Поколение]]
6. [[Автогик/3 Информационная архитектура/Структура данных/Сущности/Автопром/Карточка|Карточка]]

Сущности могут иметь краткое представление (КП) и полное (ПП).
