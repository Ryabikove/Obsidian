Сущность ПО, представляющая из себя совокупность данных о пользователе.

*Представление в БД*
Имя таблицы: account

| Имя поля      | Тип данных                 | Unique         | Not Null | Default | Описание                           |
| ------------- | -------------------------- | -------------- | -------- | ------- | ---------------------------------- |
| id            | Serial                     | PK             | +        | -       | Первичный ключ                     |
| mail          | Varchar(100)               | +              | -        | -       | Электронная почта                  |
| login         | Varchar(100)               | +              | +        | -       | Логин                              |
| phone_number  | Varchar(30)                | +              | -        | -       | Номер телефона                     |
| full_name     | Varchar (250)              | -              | -        | -       | ФИО пользователя                   |
| birth_date    | Date                       | -              | -        | -       | Дата рождения                      |
| account_photo | #TODO написать тип данных  | -              | -        | -       | Фото профиля                       |
| corporation   | Boolean                    | -              | +        | False   | Является ли аккаунт корпоративным  |
| create_at     | Timestamp without timezone | -              | +        | Now()   | Дата создания аккаунта             |
| last_login    | Timestamp with timezone    | -              | +        | Now()   | Дата последнего логина             |
| country       | Varchar(100)               | FK many to one | +        | -       | Страна проживания                  |
| city          | Varchar(100)               | FK many to one | -        | -       | Город проживания                   |
| drive_since   | Date                       | -              | -        | -       | Стаж вождения                      |
| verified      | Boolean                    | -              | +        | False   | Является ли аккаунт подтвержденным |
