Сущность ПО, представляющая из себя совокупность данных о пользователе.

Содержит несколько разделов:
- Информация о пользователе
	- Привязанные аккаунты
	- ФИО
	- Логин
	- Почта
	- Телефон
	- Город
	- Стаж вождения
	- Дата рождения
- Информация об аккаунте
	- id
	- Тип аккаунта: личный/корпоративный
	- Кошелек *в перспективе*
	- Дата регистрации
- Предпочтения
	- [[Автогик/3 Информационная архитектура/Карты/Модули/Избранное|Избранное]] авто (не путать с [[Автогик/3 Информационная архитектура/Карты/Модули/Понравившееся|понравившееся]])
	- Отношение к стилям вождения: агрессивный/спокойный/динамичный/чрезмерно аккуратный
	- Текущие авто (характеристики как у [[Автогик/3 Информационная архитектура/Структура данных/Сущности/Автопром/Карточка|карточки]])
	- Список предпочтений по характеристикам
