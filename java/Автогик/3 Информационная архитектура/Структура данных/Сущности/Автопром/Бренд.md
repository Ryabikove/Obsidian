Сущность, представляющая бренд-производитель авто.

*Представление в БД*
Имя таблицы: brands

| Имя поля              | Тип данных                | Unique         | Not Null | Default | Описание                                                           |
| --------------------- | ------------------------- | -------------- | -------- | ------- | ------------------------------------------------------------------ |
| id                    | Small serial              | PK             | +        | -       | Идентификатор                                                      |
| short_name            | Varchar(50)               | +              | +        | -       | Короткое имя                                                       |
| full_name             | Varchar(100)              | +              | +        | -       | Полное имя                                                         |
| concern_id            | Small int                 | FK many to one | -        | -       | Концерн владелец                                                   |
| country_code          | Varchar(3)                | FK many to one | +        | -       | Страна основания                                                   |
| logo                  | #TODO написать тип данных | -              | -        | -       | Логотип бренда                                                     |
| capitalization        | Big int                   | -              | -        | -       | Капитализация ($)                                                  |
| grows                 | Small int                 | -              | -        | -       | Рост (%) по отношению к периоду в 3 года                           |
| description_n_history | Text                      | -              | -        | -       | Концепция + общие принципы машиностроения, а так же история бренда |
| produced_auto         | Int                       | -              | -        | -       | Кол-во производимых единиц                                         |
| sold_auto             | Int                       | -              | -        | -       | Кол-во продаваемых единиц                                          |
