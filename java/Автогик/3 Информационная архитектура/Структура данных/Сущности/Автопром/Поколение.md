Сущность, представляющая поколение конкретной модели авто.

Имеет следующую структуру данных:
- id
- Фотографии
- Ссылка на [[Автогик/3 Информационная архитектура/Структура данных/Сущности/Автопром/Бренд|бренд]]
- Ссылка на [[Автогик/3 Информационная архитектура/Структура данных/Сущности/Автопром/Модель|модель]]
- Полное название
- Годы производства
- Оценка по общим показателям
	- комфорт салона
	- вместительность
	- динамичность
	- для семьи
	- для бездорожья
	- представительность
	- компактность
	- технологичность
	- средняя оценка по показателям
- Описание поколения + обзор
- Рекомендации по комплектациям
- Список стран, где собираются автомобили, кол-во производимых и продаваемых единиц
- Отзывы от владельцев (подтвержденные)
- Ссылки на статьи по авто

Может быть представлена в виде [[Автогик/3 Информационная архитектура/Карты/Список сущностей|списка сущностей]]
