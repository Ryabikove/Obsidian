| full_name            | Varchar(100)              | +              | +        | -       | Полное имя                  |
| photos               | #TODO написать тип данных | -              | -        | -       | Фотографии поколения        |

| year_stop            | Smallint                  | -              | -        | -       | Год снятия с производства   |
| produced_auto        | Int                       | -              | -        | -       | Кол-во произведенных единиц |
| sold_auto            | Int                       | -              | -        | -       | Кол-во проданных единиц     |
