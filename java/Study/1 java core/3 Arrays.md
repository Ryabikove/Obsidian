Списки - набор данных. Массивы могут быть реализованы различными способами в java.

[[Java/variables/атрибуты переменных/значения переменной/Массив|Массив]] - упорядоченный набор значений одинакового типа с помощью добавления \[] после указания типа данных (int\[]). 
У такой реализации списка есть ряд ==ограничений==:
- После определения количества элементов, количество не может быть изменено
- Все значения должны быть одного типа
- Все явно не указанные значения элементов такого массива будут равны значениям по умолчанию своего типа (0 для чисел и char, false для boolean и null для всех остальных)
Так же есть ряд ==преимуществ==:
- Манипуляция с данными быстрее
- Экономичный расход памяти
- Возможно работать с примитивами

Так же массив может быть многомерным (int\[]\[]), к нему применяются все правила одномерного. Можно как заранее явно указать количество строк и столбцов в многомерном, так и при добавлении значений неявно.
