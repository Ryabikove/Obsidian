ООП - объектно-ориентированное программирование - принцип, при котором все составные элементы представляют из себя объекты.  Все, кроме примитивных классов является объектом (int, short, byte, long, float, double, char, boolean, var).

Основные принципы ООП:
- [[Java/package/class/модификаторы/Инкапсуляция|Инкапсуляция]] - скрытие внутреннего состояния объекта от прямого вмешательства.
- [[Java/package/class/наследование/Наследование|Наследование]] - приобретение всех атрибутов и методов родительского класса дочерним
- [[Java/package/class/наследование/Полиморфизм|Полиморфизм]] - возможность использования объектов с одинаковым интерфейсом вне зависимости от типа объекта.
- [[Java/package/class/модификаторы/Абстракция|Абстракция]] - выделение только значимых характеристик объекта, игнорируя лишние детали.

Классы и объекты в java состоят из:
- [[Java/package/class/Атрибут|Атрибут]] - переменные, объявленные внутри класса.
- [[Java/package/class/метод/метод|Методы]] - функции, которые выполняют объекты
- [[Java/package/class/метод/конструктор|Конструкторы]] - функция, отрабатывающая в момент создания экземпляра класса
