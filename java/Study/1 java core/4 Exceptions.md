Специальные классы, позволяющий описывать ошибки, возникающие в процессе работы кода, который наследуется от других исключений или базового исключения [[Java/package/built-in/java.lang/Exception|Exception]].
Сам же класс Exception наследуется от класса Throwable.

Таким образом каждая нештатная ситуация в коде может быть описана специфическим исключением, которое даст более подробное и подходящее описание ошибки, возникшей в коде.
[[Java/package/class/Исключения/исключение|исключение]]

Для вызова исключения используется конструкция [[Java/package/class/Исключения/throw|throw new *Исключение*]].
Каждый метод, вызывающий исключения в блоке кода должен иметь в своем названии конструкцию [[Java/package/class/Исключения/throws|throws *Исключение*]]
