Коллекции - более тяжеловесные и популярные аналоги массивов.
Разделяют 3 основных и наиболее популярных коллекции:
 - [[Java/package/built-in/java/util/ArrayList|ArrayList]] - упорядоченный, изменяемый список значений
 - [[Java/package/built-in/java/util/HashSet]] - неупорядоченное множество уникальных значений
 - [[Java/package/built-in/java/util/HashMap]] - по другому словарь или карта, неупорядоченное хранилище пар ключ:значение

Все коллекции в Java живут в пакете `java.util`. Все они наследуются от интерфейса Collection. Такие объекты не могут содержать примитивы.

Каждая коллекция имеет несколько аналогов:
- List:
	- ArrayList: Быстрый доступ по индексу (как массив).
	- LinkedList: Быстрая вставка и удаление в начале/середине.
- Set:
	- HashSet: Максимальная скорость, но неупорядочен
	- LinkedHashSet: Упорядочен, но скорость ниже.
	- TreeSet: Автоматическая сортировка элементов
- Map:
	- HashMap: Максимальная скорость, но неупорядочен
	- LinkedHashMap: Упорядочен, но скорость ниже.
	- TreeMap: Автоматически сортирует по ключам.
