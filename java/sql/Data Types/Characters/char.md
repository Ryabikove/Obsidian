или character(), bpchar()
символьный тип данных строка фиксированной длины.

имеет синтаксис
 val char(n) - где n целое число в диапазоне от 1 до 10485760 включительно
 без указания длины n = 1

если в столбец данного типа внесли строку короче n, то строка будет дополнена пробелами, пока длина не станет n символов
если в столбец данного типа внести строку длиннее n символов, то возникнет ошибка и данные не запишутся, если только все лишние символы не являются пробелами (тогда они будут усечены до максимально допустимой длины).

==может хранить до 255 байт информации==

Начиная с MySQL 5.6 типы char и varchar по умолчанию используют кодировку UTF-8

==при сравнении двух значений типа character дополняющие пробелы считаются незначащими и игнорируются==

==при приведении значения character к другому символьному типу дополняющие пробелы отбрасываются==
