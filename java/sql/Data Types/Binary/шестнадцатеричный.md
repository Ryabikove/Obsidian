формат кодирования бинарного типа данных
данные кодируются двумя шестнадцатеричными цифрами на байт, первая цифра соответствует старшим 4 битам
к полученной строке добавляется префикс \\x для отличия от формата спецпоследовательности
==в некоторых контекстах обратную косую черту нужно экранировать, продублировав её==

вводимые шестнадцатеричные цифры могут быть в любом регистре
между парами цифр пробельные символы допускаются, ==но внутри пар или вначале последовательности \\x нет==

==совместим== со множеством внешних приложений и протоколов, ==преобразуется обычно быстрее==
