числовой тип
хранит приближённые числовые значения с переменной точностью
диапазон значений от 1E-307 до 1E+308 с точностью не меньше 15 десятичных цифр

попытка сохранить значения вне диапазона приведёт к ошибке

если точность вводимого числа больше допустимого оно будет округлено

==объем данных 4 байта==

==некоторые значения, которые нельзя преобразовать во внутренний формат, сохраняются приближённо, так что полученное значение может отличаться от записанного==

при попытке сохранить число, близкое к 0, но непредставимое как отличное от 0, произойдёт ошибка антипереполнения

==по умолчанию числа с плавающей точкой выводятся в текстовом виде в кратчайшем точном десятичном представлении
преобразование в кратчайший точный вид производится гораздо быстрее, чем в традиционное представление с округлением.==

так же decimal принимает специальные значения:
 - Infinity
 - -Infinity
 - NaN
