управляющая конструкция, выполняющая циклично блок кода, пока условие, указанное в конструкции истинно

существует 2 разновидности этой конструкции:
```
do {...} while();
```
- выполняет проверку после выполнения блока кода
- выполняет всегда минимум 1 итерацию
```
 while () {};
```
- выполняет проверку перед выполнением блока кода

может использоваться с ключевыми словами:
 - [[управляющие конструкции/циклы/continue|continue]]
 - [[управляющие конструкции/циклы/break|break]]
