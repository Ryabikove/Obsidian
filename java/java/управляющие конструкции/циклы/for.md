конструкция определяющая количество повторений определенного блока кода в зависимости от указанных условий

синтаксис:
```
for ([инициализация счетчика]; [условие]; [изменение счетчика]) {
	[блок кода]
}
for (int i = 1; i < 9; i++){
	System.out.printf("Квадрат числа %d равен %d \n", i, i * i);
}
```

счетчик может быть любым числовым типом

не обязательно указывать все условия при объявлении цикла:
 - счетчик можно инициализировать заранее
 - изменение счетчика можно указать внутри цикла
```
	int i = 1;
	for (; i<9;){
		System.out.printf("Квадрат числа %d равен %d \n", i, i * i);
		i++;
	}
```

бесконечный цикл for:
```
for (; ;){
    System.out.printf("Квадрат числа %d равен %d \n", i, i * i);`
}
```

внутри цикла так же можно определять сразу несколько переменных и управлять ими:
```
int n = 10;
for (int i=0; j = n - 1; i < j; i++; j--) {
	System.out.print(i\*j);
}
```

может использоваться с ключевыми словами:

 - [[управляющие конструкции/циклы/break|break]]

==foreach== - специальная версия конструкции for, предназначенная специально для перебора эл-тов массива
функционал такой же, как и в других языках
синтаксис:
```
 for (тип_данных название_переменной : контейнер) {
	actions
 }
 int[] array = new int[] {1,2,3,4,5};
 for (int i : array){
	System.out.println(i);
 }
 for (int i = 0; i < array.length; i++){
	System.out.println(array[i]);
 }
```
==для перебора многомерных массивов создаются вложенные циклы==
