управляющая конструкция, предназначенная для обработки исключений и ошибок
при возникновении исключения в блоке try управление переходит в блок catch, который обрабатывает это исключение
если блока catch нет, то выводится сообщение об необработанном исключении, дальнейшее выполнение программы останавливается
блок finally обрабатывается абсолютно всегда после всех блоков

синтаксис:
try {
	[[блок кода]]
}
catch(тип_исключения имя переменной){
	[[блок кода]]
}
finally {
	[[блок кода]]
}
если возникшее исключение не является исключением типа, указанного в блоке catch, то исключение не обработается
можно использовать несколько блоков catch для обработки разного типа исключений

блок catch и finally не обязательные, программа сможет работать и без них

для самостоятельного вызова исключений используется оператор [[java/пакет/класс/Исключения/throw|throw]]
