комментарии в java присутствуют так же, как и в любом другом языке программирования

создаются при помощи последовательности символов "\/\* какой-то текст \/\*"

могут размещаться на нескольких строках

они не несут никакой функциональный характер, а только информативный

при компиляции игнорируются
