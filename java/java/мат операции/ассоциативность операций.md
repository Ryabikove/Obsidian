когда операции имеют один и тот же приоритет, порядок вычисления определяется ассоциативностью операторов. 

в зависимости от ассоциативности есть два типа операторов:
 - левоассоциативные операторы - выполняются слева направо
	все операции, кроме префиксного инкремента и декремента
 - правоассоциативные операторы - выполняются справа налево
	 префиксный инкремент и декремент
