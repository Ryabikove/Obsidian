Специальное хранилище в памяти, позволяющее ссылать разные объекты на одну область.

String pool работает в случае, когда мы передаем одинаковое значение строки двум объектам напрямую:
```
String s1 = "Book";
String s2 = "Book"; # s1==s2
String s3 = new String("Book"); # s3!=s2 (new создает строку вне пула)
String s4 = new String("Book").intern(); #s4==s2==s1
```

В java тип данных String неизменяемый, то есть если есть строка "java", то при её форматировании в строку "Java" будет создан новый объект в памяти.

Для добавления строк в пул существует метод [[java/пакет/встроенные/java.lang/String|intern()]] класса String.
