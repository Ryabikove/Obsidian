поразрядная операция сдвига разрядов в лево или право

существует 3 разновидности сдвига:
 - "<<" - сдвиг влево.
	 4<<1 сдвигает число 4 на 1 разряд влево. 100->1000; 4->8
 - ">>" - сдвиг вправо.
	16>>1 сдвигает число 16 на 1 разряд вправо. 10000->1000; 16->8
 - ">>>" - беззнаковый сдвиг вправо
	 -8>>>2 сдвигает -8 на 2 разряда вправо. 1111 1111 1111 1111 1111 1111 1111 1000 -> 11 1111 1111 1111 1111 1111 1111 1110
