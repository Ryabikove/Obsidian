базовый встроенный класс, от которого неявно наследуются все остальные встроенные и создаваемые классы

методы:
 - [[пакет/класс/встроенные/Object/методы/equals()|equals()]] - сравнивает 2 объекта между собой
 - [[пакет/класс/встроенные/Object/методы/getClass()|getClass()]] - возвращает имя класса, к которому принадлежит экземпляр
 - [[пакет/класс/встроенные/Object/методы/hashCode()|hashCode()]] - возвращает хеш-код
 - [[пакет/класс/встроенные/Object/методы/toString()|toString()]] - возвращает строковое описание объекта
