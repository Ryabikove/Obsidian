указатель статичности, показывающий, что метод или поле независим от объекта и доступен вне своего класса. (может вызываться без создания экземпляра класса)

==особенности==:
 - статические переменные объявляются для класса, а не для объекта.
 - статическую переменную можно вызвать без создания объекта
 - вызов статического инициализатора происходит после загрузки класса и до создания самого первого объекта класса

==требования==
 - в статичных методах можно вызвать только статичные методы и использовать только статичные переменные
	обычно методы определяются как статические, когда они не затрагивают состояние объекта или/и для вызова метода нет смысла создавать экземпляр класса
