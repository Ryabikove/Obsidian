класс - основная комплексная единица кода в java, шаблон написания объекта
объект - уникальный экземпляр класса

в состав класса входят:
 - [[пакет/класс/метод/метод|методы]]
 - [[java/пакет/класс/Атрибут|атрибуты]]

синтаксис определения:
```
 *modificators* class  Class_name {
	 *block of code*
}
```
[[java/пакет/класс/модификаторы/модификатор|модификатор]]  [[java/блок кода|блок кода]]

любой объект обладает двумя основными хар-ми:
 - состояние - данные, которые хранит в себе объект
	для хранения состояния используются [[переменные/переменная|переменные]]
 - поведение - действия, которые может совершать объект
	для определения поведения используются [[пакет/класс/метод/метод|методы]]

==требования==:
 - главный класс любой программы на java ==должен== содержать метод [[main]]
 - название файла и главного класса в файле ==должны== совпадать

==особенности==:
 - для создания объекта используется оператор [[пакет/класс/new|new]]
 - при создании объекта определенного класса вызывается [[пакет/класс/метод/конструктор|конструктор]] - специальный метод [[пакет/класс/инициализация объекта|инициализирующий]] объект
 - классы могут объединяться в пакеты
 - в java присутствуют встроенные классы
 - Один класс может [[java/пакет/класс/наследование/Наследование|наследоваться]] от другого
 - помимо базовых классов есть абстрактные классы [[пакет/класс/модификаторы/abstract|abstract]]
	абстрактный класс отличается от обычного тем, что нельзя использовать его конструктор напрямую, а только через производный класс. основная задача абстрактного класса - быть унаследованным производным классом
 - класс может быть определен внутри другого класса или интерфейса
	[[пакет/класс/внутренние классы|внутренние классы]]
 - при написании класса можно задать универсальный параметр, позволяющий определять тип полей класса при создании экземпляра - обобщенный класс
	[[java/пакет/класс/обобщения/generics-обобщения|generics-обобщения]]
