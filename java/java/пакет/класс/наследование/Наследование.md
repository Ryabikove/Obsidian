Принцип объединения функционала двух классов или интерфейсов путем получения всех характеристик родительского класса или интерфейса дочерним

class Classint1 {
	Type var1;
	Type var2;
	mod method1(type var){
		action
	}
}
class Classint2 [[java/пакет/класс/наследование/extends|extends]] Classint1{
	mod Classint2(type var){
		[[java/пакет/класс/наследование/super|super]](var);
	}
}

interface inter1{}
interface inter2 extends inter1{}

для наследования интерфейса классом используется ключевое слово [[java/пакет/класс/наследование/интерфейс/implements|implements]]
inetface int1{}
class Class1 implements int1{}

==особенности==:
 - производный класс/интерфейс имеет доступ ко всем методам и полям базового класса/интерфейса, кроме определенных модификатором [[пакет/класс/модификаторы/private|private]]
 - можно запретить наследование ключевым словом [[пакет/класс/модификаторы/final|final]]
	 modificators... final class Class1 {}
- класс наследуется с помощью слова [[java/пакет/класс/наследование/extends|extends]] только от класса, а интерфейс только от интерфейса

==требования==:
 - наследование может производиться не более чем от одного класса/интерфейса
