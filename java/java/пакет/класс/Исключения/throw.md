оператор вызова исключений в процессе выполнения кода.

синтаксис:
```
throw new тип_исключения(аргументы);
```

При вызове исключения выполнение метода прекращается, как при вызове [[java/пакет/класс/метод/return|return]], но управление передается в ближайший блок [[java/управляющие конструкции/условные/try catch finally|catch]].
