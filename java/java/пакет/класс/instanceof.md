ключевое слово, использующееся для проверки принадлежит ли экземпляр к определенному классу

возвращает значения типа boolean:
 - true - если принадлежит классу, или наследует его
 - false - если не принадлежит классу

==особенности==:
 - с помощью этого слова можно осуществлять преобразование классов внутри конструкции
	val1 instanceof Class1 val2
	если val1 принадлежит классу Class1 (возвращается значение true), то создается переменная val2 класса Class1
