Класс и инструмент, позволяющий обрабатывать случаи отсутствия передаваемых параметров.

Подразумевает собой статический класс с произвольным значением внутри.
Для помещения значения внутрь используются следующие методы:
- `.of(value)` - если мы уверены, что значение есть. Если передать `null`, выбросит исключение;
- `.ofNullable(value)` - если не уверены в наличии значения. Если передать `null`, ничего не случится;
- `.empty()` - если уверены, что значения нет

Поддерживает те же методы, что и [[java/управляющие конструкции/Stream-api/Stream-api|stream-api]], что позволяет писать цепочки обработки:
- `ifPresent(Consumer)` - выполнит действие, если внутри что-то есть
- `orElse(default)` - вернет значение из класса или дефолтное, если значения нет
- `orElseGet(Supplier)` - то же, что и предыдущее, но значение по-умолчанию вычислит только при необходимости
- `orElseThrow()` - выбросит ошибку, если значения нет (можно настроить на конкретную)
- `map(Function)` - преобразует значение внутри, если такое есть

Синтаксис:
```
Object name = Optional.ofNullable(method)
	.map(Object::method)
	.map(obj -> obj.method())
	.orElse(new Object);
```
