[[java/пакет/класс/наследование/Функциональный интерфейс|Функциональный интерфейс]], позволяющий принимающий два операнда класса T и возвращающий результат вычислений того же класса.

Аналог [[java/пакет/встроенные/java.util/function/UnaryOperator|UnaryOperator]], только принимает не один операнд, а два.
Реализовывает метод - R apply(T t, U u);

Пример кода:
```
BinaryOperator<Integer> sqrt = (s,c) -> s*c; 
int l = sqrt.apply(2,3); // 6
```
