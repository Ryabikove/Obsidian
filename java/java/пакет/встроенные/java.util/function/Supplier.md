[[java/пакет/класс/наследование/Функциональный интерфейс|Функциональный интерфейс]], позволяющий получить новый объект.

Реализовывает единственный метод - T get();

Пример кода:
```
Supplier<Object> isNotEmpty = () -> new Object(); 
Object result = isNotEmpty.test("Hello"); // true
```
