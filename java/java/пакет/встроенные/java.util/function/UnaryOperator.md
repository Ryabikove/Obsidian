[[java/пакет/класс/наследование/Функциональный интерфейс|Функциональный интерфейс]], позволяющий принимающий класс T и возвращающий его же.

Аналог [[Function]], только принимаемые и возвращаемые данные совпадают по типу.
Реализовывает единственный метод - R apply(T t);

Пример кода:
```
UnaryOperator<Integer> sqrt = s -> s*s; 
int l = sqrt.apply(2); // 4
```
