более тяжеловесный аналог [[java/переменные/атрибуты переменных/значения переменной/массив|массива]], но с большей гибкостью в настройке, зачастую используется, когда нужен простой список чего либо, где важен порядок добавления. 

Зачастую используется вместо массива.
Использует [[java/пакет/класс/обобщения/generics-обобщения|обобщения]] для указания класса, который будет хранить.
