последовательность символов, указывающая на определенную область в памяти

==особенности==:
 - в качестве имени объекта (метода/класса/переменной) может выступать произвольное сочетание символов
 
==ограничение==:
 - по алфавиту - латинский
 - по набору символов - буквы любого регистра, цифры, знак подчеркивания
 - по последовательности - первый символ не цифра
 - по общему названию - не ключевое слово языка java

т.к. java регистрозависимый, то и в названии переменных учитывается регистр
