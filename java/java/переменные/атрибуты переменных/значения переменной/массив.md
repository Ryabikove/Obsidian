упорядоченный набор однотипных значений

синтаксис объявления массива:
```
int x[], x1[], x2, x3[];
int[] x, x1, x2, x3;
```
синтаксис инициализации массива:
```
 x = new int[10];
 int[] x = new int[10]; # оба массива из 10 значений
```
при подобной инициализации все элементы массива имеют [[значения по умолчанию]]
синтаксис присвоения значений:
```
 int[] nums = {1,2,3,4,5}
 int nums[] = new int[] {1,2,3,4,5}
```
обращение к конкретному эл-ту массива происходит так же, как и в других языках:
```
 int[] nums = {1,2,3,4,5};
 System.out.print(nums[0]);
```
при обращении можно заменять эл-ты по индексу (начинается с 0)
при обращении к эл-ту, выходящему за пределы массива, вызывается ошибка

массив может быть многомерным (содержать внутри другие массивы):
```
int[][] nums = {{1,2,3},{4,5,6}}
```
 происходит иначе:
  - обращение `nums[1][2]`
  - объявление `int[][] nums`
  - инициализация `int[][] nums = new int[4][]`
	здесь мы определяем только кол-во массивов внутри многомерного массива, а кол-во значений во всех внутренних
  - каждому массиву внутри многомерного можно присвоить разное кол-во значений:
```
	int[][] nums = new int[3][] # 3 массива внутри одного
	nums[0] = new int[3]; # 3 значения внутри первого
	nums[1] = new int[4]; # 4 значения внутри второго
	nums[2] = new int[1]; # 1 значение внутри третьего
```
==двойные квадратные скобки обязательны при осуществлении взаимодействия с многомерным массивом==

==нельзя инициализировать массив и присваивать ему значение одновременно
запись подобного вида приведет к ошибке==
```
int[][] nums = new int[3][3]{{ 1, 2, 3 }, { 3, 4, 5 }, { 6, 7, 8}};
```
При создании массива, выделяемая память вычисляется по формуле: размер одного эл-та \* количество эл-тов:
```
int[] arr = new int[5] #int = 4 байта * 5 эл-тов = 20 байт
```
Индекс эл-та представляет из себя смещение от начала массива, поэтому доступ к любому эл-ту массива происходит мгновенно (O(1)).

Размер массива неизменяем, для его увеличения необходимо создать новый массив и скопировать туда старые данные.
Копирование может происходить с помощью цикла for (медленно) или методов [[java/пакет/класс/встроенные/System/System|System.arraycopy()]], [[Array.copyOf()]] (быстро).

Зачастую для сортировки используются не циклы а готовые методы класса [[Arrays]].
В основном такие как:
- sort(arr) - быстрая сортировка (Dual-Pivot Quicksort)
- binarySearch(arr, key) - мгновенный поиск в отсортированном массиве
- deepToString(twoDimArr) - лучший способ распечатать многомерный массив в консоль
