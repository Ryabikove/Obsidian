примитивный тип данных переменной

ограничения:
 - по типу данных - одиночный символ в кодировке UTF-16 в диапазоне от 0 до 65535
 - по объему в памяти - <= 2 байта

заключается в одинарные кавычки, если содержит символ или его шестнадцатеричную форму:
 char a = 'a';
 char a = '\\u0066' \# символ 'f'
не заключается в кавычки, если содержит код символа в виде целого числа в 10-ой системе счисления
 char a = 102; \# символ 'f'
