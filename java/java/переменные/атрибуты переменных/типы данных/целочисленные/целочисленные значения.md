значения, представляемые в виде целого числа без дробной части

к таким значениям относятся:
 - [[переменные/атрибуты переменных/типы данных/целочисленные/int|int]]
 - [[переменные/атрибуты переменных/типы данных/целочисленные/short|short]]
 - [[переменные/атрибуты переменных/типы данных/целочисленные/byte|byte]]
 - [[переменные/атрибуты переменных/типы данных/целочисленные/long|long]]
 

==без указания типа, все целочисленные значения по умолчанию принимают тип int
 при выходе из диапазона значений int попытка компиляции закончится ошибкой. для корректной компиляции в конце целого числа необходимо добавить L или l
 long num = 2147483649l ==

значения для целочисленных переменных можно задавать не только в 10-ой системе счисления, но и в 2-ой, 8-ой и 16-ой
 для того, чтобы компилятор понимал систему счисления, перед числом добавляется приставка
 0 - 8-ая: int x=010 \# 8 в 8-ой
 0х - 16-ая: int x=0x6F \# 16 в 16-ой
 0b - 2-ая: int x=0b1101 \# 13 в 2-ой

разряды числа могут быть разделены знаком "\_":
int x = 123_123 \# 123123
