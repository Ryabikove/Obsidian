язык ооп, поддерживающий полиморфизм, [[Java/package/class/наследование/Наследование|Наследование]], статическую типизацию

==особенности==:
 - регистрозависимый
 - близкий по синтаксису к C / C++/ C\#
 - поддерживает автоочищение памяти
 - для работы требует установленный [[jdk инструмент]]
 - код транслируется в специальный байт-код, независимый от платформы
 - байт-код выполняется "виртуальной машиной" [[Java/tools/JVM|JVM]] (java virtual machine)
