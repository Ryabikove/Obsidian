для записи чисел со знаком применяется дополнительный код (two's complement) при котором старший разряд является знаковым.
старший разряд = 0, - число положительное \#  00000001 = 1, 0000 1101 = 13
старший разряд = 1 - число отрицательное \# 1111 1111 = -1, 11110011 = -13

этим старшим битом обусловлен диапазон значений чисел в каждом типе:
 byte в диапазоне от 0000 0000 = 0 до 1111 1111 = -128
 ==максимальное значение каждого числового типа меньше на 1 бит наименьшего отрицательного==
 127 = 0111 1111
 -128 = 1111 1111
