многострочный текст, облеченный в тройные кавычки
 String text = """When I was older
			 I was a sailor
			 In the open sea"""
может содержаться только в переменных типа String

альтернативный способ использовать многострочность в переменных типа String:
 String text = "But now I'm underwater,\\n"+
			 "And my skin is paler\\n"+
			 "Than it should ever be"
 "+" используется только для более наглядного отображения переноса, точно такой же результат будет, если все подстроки соединить в одну строку. основную роль выполняет управляющая последовательность \\n
