каждая переменная обладает 3 атрибутами:
 - [[Java/variables/атрибуты переменных/типы данных/тип данных|тип данных]]
 - [[Java/variables/атрибуты переменных/имя объекта|имя объекта]]
 - [[Java/variables/атрибуты переменных/значения переменной/значение переменной|значение переменной]]
