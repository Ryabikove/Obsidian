Represents definite part of memory, which stores data.

Every variable has at least 3 required attributes:
- [[Java/variables/attributes/data type/Data type|Data type]];
- [[Java/variables/attributes/Name|Name]];
- [[Java/variables/attributes/value/Value|Value]].
And 1 optional - [[Java/variables/modifiers/Modifiers|modificator]].
```
[modificator] type name = value;
```

Variables determinates by 3 types of memory allocations:
- [[Java/variables/memory allocations/Local|Local]];
- [[Java/variables/memory allocations/Instance|Instance]]; 
- [[Java/variables/memory allocations/Static|Static]].

==особенности==:
 - можно задать тип значения переменной при создании объекта
	[[Java/package/class/обобщения/generics-обобщения|generics-обобщения]]

[[Java/variables/объявление и присвоение|объявление и присвоение]]

Declaration of variable :
```
int x, x1, x2, x3;
```
