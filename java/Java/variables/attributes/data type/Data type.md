Defines restrictions of value and memory in [[Java/variables/Variable|variable]].

Data type has 2 main types:
- primitive - entity, which stores in variable directly and has hight speed;
- linked - usual [[Java/package/class/Class|classes]] which stores link/address to memory area with data.

Built-in *primitive* data types:
 - [[Java/variables/attributes/data type/primitive/integer/Int|int]];
 - [[Java/variables/attributes/data type/primitive/Var|var]];
 - [[Java/variables/attributes/data type/primitive/Boolean|boolean]];
 - [[Java/variables/attributes/data type/primitive/Char|char]];
 - [[Java/variables/attributes/data type/primitive/float/Float|float]];
 - [[Java/variables/attributes/data type/primitive/float/Double|double]];
 - [[Java/variables/attributes/data type/primitive/integer/Long|long]];
 - [[Java/variables/attributes/data type/primitive/integer/Byte|byte]];
 - [[Java/variables/attributes/data type/primitive/integer/Short|short]].

Every primitive data type has wrapped class.

Before assigning a value to a variable of the same type, both variables must be converted to the same type.
