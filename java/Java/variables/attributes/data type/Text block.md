Text with multiple rows inside triple quotes.
```
String text = """When I was older
	I was a sailor
	In the open sea"""
```

Stores in class [[Java/package/built-in/java/lang/String|String]].

Alternative way to create multi-row string:
```
String text = "But now I'm underwater,\n"+
	"And my skin is paler\n"+
	"Than it should ever be"
```

\\n is a [[Java/constructions/Control sequence|control sequence]].
