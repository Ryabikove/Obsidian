Primitive data type, which contains one character.

Restrictions:
- of value - single character in UTF-16;
- of memory - <= 2 bytes.

Surruonded by single quotes when contain character or hexadecimal code:
```
char a = 'a';
char a = '\\u0066' \# символ 'f'
```
Can be written without quotes if it is in decimal integer form.
```
char a = 102; \# символ 'f'
```
