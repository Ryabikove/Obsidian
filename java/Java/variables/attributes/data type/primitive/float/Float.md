Primitive data type which contains [[Java/variables/attributes/data type/primitive/float/Floating numbers|floating number]] value.

Restrinctions:
- of value - floating numbers in range from -3.4\*10^38 to 3.4\*10^38;
- of memory - <= 4 bytes.
 
You should add "f" char in the end of value:
```
float num = 30.6f
```

It has [[Java/variables/attributes/data type/Default value|default value]].
