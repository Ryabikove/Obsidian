Primitive data type which contains [[Java/variables/attributes/data type/primitive/float/Floating numbers|floating number]] value.

Restrinctions:
- of value - floating numbers in range from ±4.9\*10^-324 to ±1.7976931348623157\*10^308;
- of memory - <= 8 bytes.
