Values represented as a number with a factorial part, where the integer part is separated from the factorial. 
Character which separates parts depends on localization:
- English - ".";
- Russian - ",".

There are two built-in data types:
 - [[Java/variables/attributes/data type/primitive/float/Double|Double]]
 - [[Java/variables/attributes/data type/primitive/float/Float|Float]]

==All floating values without data types by default are double==

If there are not enought memory in float for value, variable would be converted by java to double, unlike with long and int. But these variables would take different space in memory and executed in different ways.
