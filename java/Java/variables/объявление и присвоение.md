объявление переменной - процесс объявления имени блока данных в памяти системы
присвоение значения - процесс инициализации переменной, когда в объявленный блок памяти записываются данные

==требования==:
 - нельзя присваивать значение раньше объявления
	присвоение значения без объявления или раньше него вызовут ошибку и остановят исполнение кода

==особенности==:
 - объявление и присвоение значения переменной можно описывать в одной инструкции
 - можно объявлять и присваивать значение сразу нескольким переменным в одной инструкции
 - в одной программе может существовать несколько инструкций присвоения значений переменной, меняя значение переменной по ходу исполнения программы

синтаксис объявления и присвоения значений различается от вида по значениям.
