ключевое слово, которое может использоваться только в теле цикла for или while
при итерации цикла, если встречается это слово, то цикл прерывает исполнение кода и переходит к следующей итерации

обычно находится внутри конструкции if или в теле другого условного выражения

for/while () {
	if (){
		continue
	}
}

Оператору можно передать [[Java/constructions/управляющие конструкции/циклы/метки|метку]], которую заранее выставили, тем самым прервав итерацию не текущего цикла, а одного из внешних.
