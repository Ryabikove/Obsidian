ключевое слово, которое может использоваться только в теле цикла for или while, а так же в условной конструкции switch
при итерации цикла, если встречается это слово, то цикл прерывает исполнение кода и завершается

обычно находится внутри конструкции if или в теле другого условного выражения

for/while () {
	if (){
		break;
	}
}

Оператору можно передать [[Java/constructions/управляющие конструкции/циклы/метки|метку]], которую заранее выставили, тем самым прервав цикл не текущий, а один из внешних.
