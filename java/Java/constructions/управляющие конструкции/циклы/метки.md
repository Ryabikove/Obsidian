Метка - набор символов, определяющий цикл. Используется совместно с операторами [[Java/constructions/управляющие конструкции/циклы/break|break]] и [[Java/constructions/управляющие конструкции/циклы/continue|continue]] для возможности управления внешними циклами.

Правила именования метки аналогичны правилам именования переменных.
Синтаксис:

point1:
for (...) {
	point2:
	for (...){
		break/continue point1;
	}
}
