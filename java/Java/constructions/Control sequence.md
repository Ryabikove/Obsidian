специальная последовательность символов, обладающая определенным функционалом, которая воспринимается иначе, а не как последовательность символов

может содержаться только в переменной типа [[Java/package/built-in/java/lang/String|String]]

список последовательностей:
- \\n - перенос строки

своей сутью схожи со [[Java/package/built-in/java/lang/System/объекты/out/методы/printf/спецификатор|спецификаторами]]
