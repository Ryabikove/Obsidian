операция присваивающая значение переменной исходя из возвращаемого значения условной операции

синтаксис:
 \[первый операнд - условие] ? \[второй операнд] : \[третий операнд]
 var var1 = var2\<var3? (var2+var3) : (var2-var3)

условие = true -> возвращается 2 операнд
условие = false -> возвращается 3 операнд
