метод объекта [[Java/package/built-in/java/lang/System/объекты/out/out|out]] класса [[Java/package/built-in/java/lang/System/System|System]]

функционал: вывод в консоль некоторого значения с последующим переводом курсора консоли на следующую строку.
	System.out.println("Hello world!");
	System.out.println("Hello world...");
	# Hello world!
	# Hello world...

поддерживает базовые операции со строками (конкатенацию)
	int x=5;
	int y=6;
	System.out.println("x=" + x + ";  y=" + y);
