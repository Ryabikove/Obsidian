функционал: вывод в консоль некоторого значения с форматированием.
	x=5;
	x=6;
	System.out.printf("x=%d; y=%d \\n", x, y);
	# x=5; y=6

символы, начинающиеся с % обозначают спецификаторы, вместо которых подставляются аргументы согласно последовательности
 String name =` `"Tom";
 int age = 30;
 float height = 1.7f;
 System.out.printf("Name: %s  Age: %d  Height: %.2f \n", name, age, height);
