Класс, реализующий тип данных "строка"

ограничения:
 - по типу данных
 - по объему в памяти
 
заключается в двойные кавычки:
 String a = "a";
 
==начиная с 15 версии java поддерживает текстовые блоки==

==может содержать управляющую последовательность==
[[Java/variables/атрибуты переменных/типы данных/строковые и символьные/управляющие последовательности|управляющие последовательности]] [[Java/variables/атрибуты переменных/типы данных/строковые и символьные/текстовый блок|текстовый блок]]

Содержит методы:
- strip() - удаляет пробелы справа и слева строки
- repeat(n) - возвращает строку повторенную n раз
- split('n') - возвращает разделенную на массив строку по символам n
- join("n", 'a', 'b') - склеивает строки a и b, разделив символом n
- intern() - возвращает ссылку на строку из пула [[Java/memory/String pool|String pool]], если строки в пуле нет, он её создаст
