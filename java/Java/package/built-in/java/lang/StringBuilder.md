Встроенный класс, позволяющий производить операции с данными типа String без порождения лишних объектов в памяти.

Представляет из себя массив char\[], который дописывает символы по мере добавления.

Не применим в случаях использования многопоточности. Для этого используется более медленный аналог [[Java/package/built-in/java/lang/StringBuffer|StringBuffer]]

Обладает методами:
- append("n") - добавляет в конец списка элементы строки n
