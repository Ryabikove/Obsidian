Более тяжеловесный аналог [[Java/variables/атрибуты переменных/значения переменной/Array|массива]], но с большей гибкостью в настройке, зачастую используется, когда нужен простой список чего либо, где важен порядок добавления. 

Зачастую используется вместо массива.
Использует [[Java/package/class/обобщения/generics-обобщения|обобщения]] для указания класса, который будет хранить.

Наследуется от [[Java/package/built-in/java/util/List|List]]
