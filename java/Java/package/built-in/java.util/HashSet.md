Класс, реализующий неупорядоченную коллекцию уникальных элементов.
Имеет структуру хеш-таблиц

Проигнорирует попытку добавить уже существующий элемент.
Скорость проверки на уже существующий элемент О(1) благодаря структуре.
Использует [[Java/package/class/обобщения/generics-обобщения|обобщения]] для указания класса, который будет хранить.

Наследуется от [[Java/package/built-in/java.util/Set|Set]]
