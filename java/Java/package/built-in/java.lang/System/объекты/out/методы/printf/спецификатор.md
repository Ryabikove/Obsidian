особая последовательность символов, начинающаяся с %

различия между спецификаторами заключаются в типе данных, который может быть подставлен вместо них

спецификаторы:
 - %x - для подстановки шестнадцатеричных чисел
 - %f - для вывода чисел с плавающей точкой
 - %e - для подстановки чисел в экспоненциальной форме (1.3e+01)
 - %c - для подстановки одиночного символа
 - %s - для подстановки строки
 - %d - для подстановки целогочисленных значений
