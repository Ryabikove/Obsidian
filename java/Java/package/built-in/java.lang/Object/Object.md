базовый встроенный класс, от которого неявно наследуются все остальные встроенные и создаваемые классы

методы:
 - [[Java/package/built-in/java.lang/Object/методы/equals()|equals()]] - сравнивает 2 объекта между собой
 - [[Java/package/built-in/java.lang/Object/методы/getClass()|getClass()]] - возвращает имя класса, к которому принадлежит экземпляр
 - [[Java/package/built-in/java.lang/Object/методы/hashCode()|hashCode()]] - возвращает хеш-код
 - [[Java/package/built-in/java.lang/Object/методы/toString()|toString()]] - возвращает строковое описание объекта
