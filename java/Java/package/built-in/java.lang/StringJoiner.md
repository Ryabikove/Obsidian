Встроенный класс, позволяющий производить операции конкатенации списков строк с разделителями.

```
StringJoiner sj = new StringJoiner("; ", "Авторы: ", ";");
for (String author : authors) {
	sj.add(author);
}
```
Обладает методами:
- add("n") - добавляет в конец строки подстроку n, если элемент не последний, разделяет его со следующим разделителем
