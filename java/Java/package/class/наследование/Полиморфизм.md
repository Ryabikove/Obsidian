Способность программы использовать объекты с одинаковым интерфейсом без информации о конкретном типе этого объекта.

То есть у различных классов может быть один метод, но результат выполнения метода может отличаться.

Полиморфизм тесно связан с [[Java/package/class/наследование/Наследование|наследованием]].
