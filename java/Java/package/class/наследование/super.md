ключевое слово, позволяющее обращаться к полям и методам базового класса из производного:

class Class1 {
	Type var1;
	Type var2;
	mod method1(type var){
		action
	}
}
class Class2 extends Class1{
	mod Class2(type var){
		super(var);
	}
}

==требования==:
 - вызов конструктора базового класса должен идти первым в конструкторе производного класса
 - если конструктор базового класса совершает какие то действия или требует какие-либо данные на вход, его необходимо вызывать в производном, даже если это единственное действие производного
