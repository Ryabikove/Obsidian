ключевое слово, позволяющее классу [[Java/package/class/наследование/Наследование|наследовать]] интерфейс
для применения нескольких интерфейсов они все перечисляются через запятую:
синтаксис:
```
interface int1{
	type method1();
}

class Class1 implements int1,int2,int3...{}
```
