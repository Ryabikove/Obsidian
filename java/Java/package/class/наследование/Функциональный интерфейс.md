Интерфейс, в котором объявлен ровно один абстрактный метод.
Чертеж или тип данных для лямбда-выражения.
Компилятор сам догадывается, какой интерфейс вы реализуете, основываясь на действии и возвращаемом значении.

Популярные встроенные функциональные интерфейсы:
- [[Java/package/built-in/java.util/function/Predicate|Predicate<T>]] - проверяет условия и возвращает тип boolean
- [[Java/package/built-in/java.util/function/Consumer|Consumer<T>]] - выполняет действие и ничего не возвращает
- [[Java/package/built-in/java.util/function/Function|Function<T,R>]] - выполняет действие с T, возвращая тип R
- [[Java/package/built-in/java.util/function/Supplier|Supplier<T>]] - поставляет объект

Такие интерфейсы помечаются аннотацией `@FunctionalInterface`
