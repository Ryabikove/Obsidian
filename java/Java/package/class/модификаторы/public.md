модификатор доступа, указывающий, что класс или член класса общедоступен.
поля и методы такого класса или метода доступны другим классам из текущего и внешних пакетов

==требования==:
 - только один класс в каждом файле может иметь модификатор public

==особенности==:
 - нет ограничений на кол-во публичных методов
