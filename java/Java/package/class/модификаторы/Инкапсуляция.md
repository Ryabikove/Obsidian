Инкапсуляция - это механизм скрытия внутреннего состояние объекта от прямого вмешательства. 
В java это выражается тем, что мы не можем взаимодействовать с некоторыми объектами, методами или классами напрямую, а только через другие методы или классы.

Для определения уровня доступа используются [[Java/package/class/модификаторы/модификатор|модификаторы]].
Недоступные напрямую объекты бывают частично доступны через "геттеры" и "сеттеры".
