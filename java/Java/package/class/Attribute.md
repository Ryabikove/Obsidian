Переменная, объявленная внутри класса и хранящаяся в памяти.

Атрибуты могут иметь модификаторы доступа:
- [[Java/variables/modificators/private|private]] - доступ только внутри класса, напрямую нельзя обратиться
- [[Java/variables/modificators/public|public]] - доступна при обращении напрямую

Атрибут может иметь так же следующие модификаторы:
- [[Java/variables/modificators/static|static]] - атрибут с модификатором объявляется до появления экземпляра и является общим для всех экземпляров
- [[Java/variables/modificators/final|final]] - атрибут является константой и присвоение значений может произойти только единожды

Атрибуты хранятся в памяти до тех пор, пока существуют экземпляры класса, внутри которого они объявлены.

Атрибут, объявленный внутри метода, хранится в памяти только во время работы этого метода, после память очищается.
