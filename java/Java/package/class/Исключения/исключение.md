Специальный класс, описывающий тип ошибки или саму ошибку, возникающую в ходе работы программы

существует множество различных типов исключений:
 - [[Java/package/built-in/java/lang/Exception|Exception]] - базовый тип исключений и встроенный класс

Чтобы создать свое исключение, ему нужно наследоваться от другого исключения.
Исключение может проверяться при компиляции (checked), для этого нужно наследовать его от класса Exception. Иначе проверка производиться не будет (unchecked).

Если в блоке кода явно прописан вызов исключения, то метод, которому принадлежит блок, должен предупреждать о вызове исключения через слово [[Java/package/class/Исключения/throws]].
Если исключение вызывается в блоке [[Java/constructions/управляющие конструкции/условные/try catch finally|try]] и обрабатывается в catch, то конструкция throws не нужна.
