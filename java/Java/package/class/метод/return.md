ключевое слово-оператор, позволяющий возвращать значения из метода и выходить из метода

возвращать можно литеральное выражение, значение переменной или какого-то сложного выражения
помимо оператора return в строке описания метода вместо void необходимо указать тип данных возвращаемого значения
static int method(){
 return a+b/a;
}
если указанное в описании метода значение не соответствует возвращаемому, то произойдет ошибка

если необходимо только выйти из метода, но ничего не возвращать, тогда заменять void нет необходимости
