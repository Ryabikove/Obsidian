способ использования методов, расположенных, внутри одного класса с одним и тем же именем, но разным кол-вом параметров и/или типов данных

пример:
public class Program{
	public static void main(String\[] args) {
		System.out.println(sum(2,3));
		System.out.println(sum(4.5,2.3));
		System.out.println(sum(4,3,7));
	}
	static int sum(int x, int y){
		return x+y;
	}
	static double sum(double x, double y){
		return x+y;
	}
	static int sum(int ...x){
		int res=0;
		for (int i : x){
			res+=i;
		}
		return res;
	}
}

==особенности==:
 - при вызове метода, система выберет наиболее подходящую версию из указанных
 - на перегрузку влияют только типы и кол-во принимаемых параметров, тип возвращаемого значения, указанный в описании метода никак не влияет.
