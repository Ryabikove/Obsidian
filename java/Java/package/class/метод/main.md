основной метод главного класса программы, с которого начинается выполнение инструкций

==особенности==:
 - такой метод может быть в любом классе файла, а не только в главном

==требования==:
 - этот метод обязательно должен присутствовать в главном классе файла, иначе возникнет ошибка, потому что
	при запуске приложения [[jvm]] ищет в главном классе программы метод под названием main, после обнаружения запускает его
