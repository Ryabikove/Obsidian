параметры, передаваемые методу при его вызове
указываются в скобках в строке описания метода вместе с типом параметров
 static void method(int x, float y)

==требования==:
 - при вызове метода важно соблюдать позиционность параметров и их тип
 - при передаче нескольких обязательных параметров и одного или более необязательных, необязательные указываются в конце:
	method( String name, int age, String ...childrens)

==особенности==:
 - если тип может быть преобразован до необходимого автоматически, то его соблюдение не обязательно
 - для принятия методом нефиксированного числа параметров от 0 до \* используется троеточие перед названием параметра, такой параметр не обязателен, или их может быть различное кол-во, такие параметры принимаются как массив данных и обрабатывать его необходимо так же: 
	method (type ...name) 
	[[Java/variables/атрибуты переменных/значения переменной/Array|Array]]
 - при передаче объекта в качестве значения передается копия ссылки на область в памяти, где расположен объект
