импорт классов и пакетов в проекты позволяет использовать их функционал в различных проектах

для импорта классов и пакетов так же используется директива-ключевое слово import (пишется после package, если такая строка имеется):
 import \*полное имя класса или пакета\*;

==особенности==:
 - чтобы включить все классы пакета используется символ \*:
	import java.util.\*;
- есть возможность использовать классы из пакета не импортируя их полностью в программу - прописывать полный путь до класса:
	java.util.Scanner in = new java.util.Scanner(System.in);
	==вышеуказанную запись необходимо использовать в случае, когда используются два одинаковых класса из разных пакетов==


==статический импорт== - особая форма импорта, при которой есть возможность использовать методы импортируемых классов, без обращения к самим классам.
 import [[Java/variables/modifiers/static|static]] java.lang.System.\*;
 out.print(String);

==Исключение== - классы из [[java.lang]] - они импортируется автоматически
