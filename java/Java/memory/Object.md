The main entity in java and part of memory.

Object is always an instance of [[Java/package/class/Class|Class]].

Object has 2 properties:
- state - data, which is stored in object ([[Java/variables/Variable|variable]])
- behaviour - actions, which can be executed ([[Java/package/class/метод/Method|methods]])

*Object must be initialized*
Initialization is a process of allocation a memory area for a unique instance of class.
Keyword [[Java/package/class/New|new]] is used to create a new class.

Object can be represented as:
- [[Java/package/class/Class|class]];
- [[Java/variables/attributes/value/Array|array]].

Variables with [[Java/variables/attributes/data type/Data type|primitive data types]] are *not* objects.
