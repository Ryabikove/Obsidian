Algorithms are methods of manipulating data to most efficiently transfer it from state A to state B.

Every algorithm has speed, to count speed "[[Java/operations/Big O notation|Big O notation]]" is used. There are all speed coefficients from the fastest to the slowest:
- O(1) - constant time - no matter how many data does algorithm have, time always be the same 
- O(log N) - logarithmic
- O(N) - linear time - time scales as data
- O(N \* log N)
- O(N\*\*2) - quadratic
- O(N\*\*3) - cubic
- O(2\*\*N) - exponential
- O(N!) - factorial

They include topics such as:
- Sorting
	- [[Java/algorithms/sorting/Bubble sort|Bubble sort]] - swap adjoining elements in loop;
	- [[Java/algorithms/sorting/Selection sort|Selection sort]] - ;
	- [[Java/algorithms/sorting/Merge sort|Merge sort]] - devide elements until it be sorted and then unite in sorted arrays;
- Value search
- Finding the shortest path, etc.
