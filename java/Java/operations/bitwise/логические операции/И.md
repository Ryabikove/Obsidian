логическое И или логическое умножение - знак амперсанд "&"
операция проводится поразрядно:
 - числа переводятся в двоичную форму
 - их двоичная форма перемножается поразрядно
	 если оба разряда равны 1, то операция возвращает 1, иначе 0
	 (2&5) = 0 \# 101&010 (1\*0,0\*1,1\*0) = 000

при использовании в условных операциях:
 - true -  если обе операнда true, в остальных случаях - false

может иметь как двойную, так и одинарную запись ("&" или "&&"):
 - при одинарной записи вычисляются оба операнда
 - при двойной записи правый операнд вычисляется, только если левый = true
