логическое ИЛИ или логическое сложение - знак прямой слеш "|"
операция проводится поразрядно:
 - числа переводятся в двоичную форму
 - их двоичная форма складывается поразрядно
	 если один из разрядов равен 1, то операция возвращает 1, иначе 0
	 (2&5) = 7 \# 101&010 (1\*0,0\*1,1\*0) = 111

при использовании в условных операциях:
 - true -  если один из операндов true, в остальных случаях - false

может иметь как двойную, так и одинарную запись ("|" или "||"):
 - при одинарной записи вычисляются оба операнда
 - при двойной записи правый операнд вычисляется, только если левый = false
