логическое НЕ или логическое отрицание - в логических операциях знак тильда "~"
операция проводится поразрядно:
 - число переводится в двоичную форму
 - его двоичная форма меняется поразрядно
	 заменяет значение разряда на противоположное
	 (~5) = -6 \# ~0000 0101 = 1111 1010 = -6

при использовании в условных операциях знак "!":
 - true -  если операнд false, в остальных случаях - true

==может проводиться только над одним числом==
