логическое ИЛИ или XOR - знак "^"
операция проводится поразрядно:
 - числа переводятся в двоичную форму
 - их двоичная форма исключается поразрядно
	 если значения разрядов отличаются, то возвращается 1, иначе 0
	 (9^5) = 12 \# 1001^0101 (1\*0,0\*1,0\*0,1\*1) = 1100 

при использовании в условных операциях:
 - true -  если только один из операндов true, в остальных случаях - false
