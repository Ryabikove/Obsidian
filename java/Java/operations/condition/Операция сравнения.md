Условная операция сравнивающая операнды и возвращающая значение типа [[Java/variables/атрибуты переменных/типы данных/boolean|boolean]].

Сравнение для строк, коллекций и массивов происходит поссылочно, а не по значению. Для сравнения по содержанию используется метод класса Object [[Java/package/built-in/java/lang/Object/методы/equals()|equals()]].

операции сравнения:
 - "\=\=" - true - если равны, false - если не равны
 - "!=" - true - если не равны, false - если равны
 - "<" - true - если операнд слева меньше, false - если операнд справа меньше
 - ">" - true - если операнд справа меньше, false - если операнд слева меньше
 - ">=" - true - если операнд справа меньше или равен, false - если операнд слева меньше
 - "<=" - true - если операнд слева меньше или равен, false - если операнд справа меньше
