Concept which describe way how to mesure [[Study/2 algorithms and data structures/Algorithms|algorithms]] speed.
There are few big O times (from the fastest to the slowest):
- O(1) - constant time - no matter how many data does algorithm have, time always be the same 
- O(log N) - logarithmic
- O(N) - linear time - time scales as data
- O(N \* log N)
- O(N\*\*2) - quadratic
- O(N\*\*3) - cubic
- O(2\*\*N) - exponential
- O(N!) - factorial
