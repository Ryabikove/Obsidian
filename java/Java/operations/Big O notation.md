Concept which describe way how to mesure [[Study/2 algorithms and data structures/Algorithms|algorithms]] speed.
There are few big O times (from the fastest to the slowest):
1. O(1) - constant time - no matter how many data does algorithm have, time always be the same 
2. O(log N) - logarithmic
3. O(N) - linear time - time scales as data
4. O(N \* log N)
5. O(N\*\*2) - quadratic
6. O(N\*\*3) - cubic
7. O(2\*\*N) - exponential
8. O(N!) - factorial
