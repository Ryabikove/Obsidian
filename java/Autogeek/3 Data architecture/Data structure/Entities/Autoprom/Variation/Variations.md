Сущность, представляющая вариацию поколения авто.

Нет краткого и полного представлений. Отображается внутри полного представления [[Autogeek/3 Data architecture/Структура данных/Сущности/Автопром/Поколение/Поколение|поколения]].

*Представление в БД* - [[Autogeek/3 Data architecture/Структура данных/Сущности/Автопром/Вариация/Таблицы/Variations|variations]]

Имя таблицы: other_spec

| Имя поля          | Тип данных | Unique         | Not Null | Default | Описание               |
| ----------------- | ---------- | -------------- | -------- | ------- | ---------------------- |
| id                | Serial     | PK             | +        | -       | Идентификатор          |
| variations_id     | Int        | FK one to many | +        | -       | Идентификатор вариации |


Имя таблицы: wheel_orientation_type

| Имя поля                    | Тип данных | Unique | Not Null | Default | Описание                              |
| --------------------------- | ---------- | ------ | -------- | ------- | ------------------------------------- |
| id                          | bytea      | +      | -        | -       | Идентификатор                         |
| wheel_orientation_type_name | String     | +      | -        | -       | Название расположения рулевого колеса |
Имя таблицы: eco_class_type

| Имя поля            | Тип данных | Unique | Not Null | Default | Описание                       |
| ------------------- | ---------- | ------ | -------- | ------- | ------------------------------ |
| id                  | bytea      | +      | -        | -       | Идентификатор                  |
| eco_class_type_name | String     | +      | -        | -       | Название экологического класса |

Общие показатели:
- комфорт салона
- вместительность
- динамичность
- для семьи
- для бездорожья
- представительность
- компактность
- технологичность
- средняя оценка по показателям

Таблица характеристик собирается из всех строк таблиц: body_spec, engine_spec, susp_brake_spec, other_spec.

Не может быть представлена в виде [[Autogeek/3 Data architecture/Карты/Модули/Дополнительные/Список сущностей|списка сущностей]]
