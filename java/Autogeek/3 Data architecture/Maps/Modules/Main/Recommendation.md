Окно с перечнем рекомендованных моделей авто.

Принципы:
- Представляет из себя [[Autogeek/3 Data architecture/Карты/Модули/Дополнительные/Список сущностей|список сущностей]]: моделей и поколений.
- Если пользователь не авторизованный, то содержит информацию из [[Autogeek/3 Data architecture/Карты/Модули/Основные/Популярные|списка популярных]].
- Рекомендации собираются в приоритете из следующих:
	- Избранное.
	- Просмотры (последние 3 месяца).
- Список упорядочен автоматически, изменить нельзя.

Переход к модулю осуществляется:
- На сайте:
	- Через [[Autogeek/3 Data architecture/Карты/Модули/Дополнительные/Основное меню|основное меню]]
- В телеграм-боте:
	- С помощью команды "/recomended"
