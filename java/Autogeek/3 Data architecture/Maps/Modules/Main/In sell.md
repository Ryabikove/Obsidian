Окно, содержащее [[Autogeek/3 Data architecture/Карты/Модули/Дополнительные/Список сущностей|список сущностей]], которые пользователь добавил в продукт.

Принципы:
- Задумывается как модуль, позволяющий добавлять сущности с целью продажи.
- В список могут быть добавлены только [[Autogeek/3 Data architecture/Структура данных/Сущности/Автопром/Offer/Offer|карточки]]
- Нельзя добавить сущность в список если аккаунт не является подтвержденным

Переход к модулю осуществляется:
- На сайте
	- Через вспомогательное меню - доступно, только если пользователь прошел авторизацию
	- Через модуль продукта "[[Autogeek/3 Data architecture/Карты/Модули/Основные/Об аккаунте|об аккаунте]]" 
- В телеграм-боте:
	- Вызывается командой "/in_sale" - доступно, только если пользователь прошел авторизацию
	- Через модуль продукта "[[Autogeek/3 Data architecture/Карты/Модули/Основные/Об аккаунте|об аккаунте]]"
