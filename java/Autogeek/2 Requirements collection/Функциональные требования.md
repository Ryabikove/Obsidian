Продукт должен:
1. Иметь базу данных по машинам и их составляющим в виде общего списка
2. Иметь возможность фильтровать авто по характеристикам
3. Позволять искать сущности базы (концерн, марку, модель, конфигурацию) по этой базе данных
4. Осуществлять сбор авто со схожими характеристиками и составляющими в список по фильтрам или референсному авто
5. Собирать таблицу хар-тик группы авто для наглядного сравнения и выстраивать сравнительные списки
6. Иметь актуальную информацию о рыночных ценах авто и потенциальных источниках приобретения
7. Личный кабинет для пользователей с сохранением избранных авто и фильтров.

Хотелось бы, чтобы продукт:
- Содержал раздел с тестированием, по результатам которого выводил список подходящих авто
- Содержал чат с ИИ ботом, позволяющим консультировать как по вопросу подбора авто, так и по работе приложения
- Калькулятор содержания авто, растаможки.
