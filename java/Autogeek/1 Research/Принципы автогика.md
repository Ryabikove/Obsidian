1. Строгий контроль для карточек:
	1. Содержимое карточки должно в полной мере описывать и соответствовать указанному конкретному авто.
		- Повторная продажа авто спустя год после первичной будет указываться в карточке автоматически для предупреждения потенциальных покупателей
		- Будет реализован механизм на автоматический поиск дубликатов карточек
		- Описание поколения или модели в целом будет считаться несоответствующим
		- Описание продавца/организации и указание способов связи с водителем запрещены
		- Данные об авто будут дополняться из ПТС, ручное редактирование данных, заполняемых от туда запрещено
	2. Вне зависимости от типа продавца будет контролироваться его рыночная активность.
		- Для частных лиц высокая активность может приводить к пометке продавца для предупреждения потенциальных покупателей
2. Актуальная информация об авто и их производителях
	1. Поиск проверенных источников (желательно сами автопроизводители)
	2. Сотрудничество с автосервисами для получения актуальной информации по рекомендациям
