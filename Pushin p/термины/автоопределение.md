свойство атрибута, определяющее то, каким образом будет происходит его заполнение и может ли пользователь взаимодействовать с ним напрямую

разделяется на:
 - автоопределение - значение задается системой автоматически, пользователь не может задать произвольное, пользователь не может с ним взаимодействовать
 - мануальное определение - задается пользователем произвольно, пользователь может с ним взаимодействовать
	имеют только атрибуты с тегом отображаются свойства [[термины/отображаемость|отображаемость]])
