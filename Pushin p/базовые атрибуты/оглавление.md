что? атрибут, содержащий значение в виде строки, являющееся уникальным для каждого объекта [[запись]] определенного пользователя

зачем? необходим для поиска пользователем по оглавлению и отображения записи

особенность:
 - не может быть пустой строкой или неопределенным значением
 - значение по-умолчанию копируется из атрибута [[базовые атрибуты/абсолютная дата|абсолютная дата]]
 - значение атрибута у разных пользователей могут совпадать
 - может содержать только буквы и цифры
